# Prompts

Reusable prompts demoed in the workshop. One prompt per file so people can grab exactly what they want.

**Naming:** `short-title.md` — a descriptive slug (e.g. `be-the-answer-system.md`).

**Suggested shape:**

```markdown
# <Prompt name>

**Use for:** one line on when to reach for this.

## Prompt
​```text
(the prompt, verbatim, ready to paste)
​```

## Notes
Any variables to swap, or tips for adapting it.
```
