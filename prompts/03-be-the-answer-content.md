# Prompt 3 — Be the Answer

**Use for:** turning one real customer question into content that *answers* it — matched to the stage
they're in, and written in your voice.

## Prompt

```text
You write content that is genuinely helpful — the answer to a real question,
not an ad. You never make hollow claims and you never sound like generic AI.

My brand context: [PASTE YOUR BRAND CONTEXT FROM PROMPT 4 — or skip if you don't have it yet]

The customer question I want to answer: [PASTE ONE QUESTION FROM PROMPT 2]
The journey stage this question belongs to: [AWARENESS / ACQUISITION / ADOPTION / ADVOCACY]
The channel: [LinkedIn / Instagram / X / blog / email]

Write a post that IS the answer to that question. Rules:
- Be genuinely useful first. Earn the attention.
- Match the stage: awareness = introduce the problem + that a solution exists;
  acquisition = help them decide; adoption = help them succeed; advocacy = make sharing easy.
- Sound like a real person, not a brand shouting. No hype. No fake claims.
- Fit the channel's length and style.
- End with a natural next step, not a hard sell.

Then give me 2 alternative opening lines (hooks) so I can pick the strongest.
```

## Notes
- The test for every post: *is this the answer to a real question, at the stage my customer is in?*
  If not, don't post it.
- Editing is easier than writing. Let AI give you the draft, then make one line unmistakably yours.
