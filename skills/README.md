# Skills

Claude skills shown in the workshop, ready for participants to drop into their own setup.

Each skill lives in its own subfolder with a `SKILL.md`:

```
skills/
└── my-skill/
    └── SKILL.md
```

**To use one:** copy the skill's folder into your project's `.claude/skills/` (or `~/.claude/skills/` for a personal, always-available skill), then invoke it with `/my-skill`.

See the [Claude Code skills docs](https://docs.claude.com/en/docs/claude-code) for the `SKILL.md` format.
