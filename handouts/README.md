# Handouts

One Markdown file per teachable moment — something a participant can take home and recreate later.

**Naming:** `NN-short-title.md` (e.g. `01-your-first-prompt.md`) so they sort in teaching order.

**Suggested shape for each handout:**

```markdown
# <What they'll be able to do>

**When to use this:** one line.

## Steps
1. ...
2. ...

## The thing to copy
(prompt / command / snippet)

## Why it works
A sentence or two.
```

See [`example-handout.md`](example-handout.md) for a filled-in template.
